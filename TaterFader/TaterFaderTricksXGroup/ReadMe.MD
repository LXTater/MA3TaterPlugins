TaterFaderXGroups - V0.1

NOTE: This is a first version, early release. It has not been fully tested. Please test before operating in a show environment.

1 - Run Plugin

2 - Set the Settings

Choose an Empty Executor

High Value: The highest value that you want XGroup to be (Fader 100%)

Low Value: The Lowest Value that you want XGroup to be (Fader 0%)

MATricks Preset: Choose the MATricks preset, the plugin will only update XGroup information so you can have other information in the preset.

StepSize: The amount that you want the plugin to step, if set to 1 it will go from every rounded value between low value and high value, if 2, it will step by 2, etc.

3 - Keep Plugin running.

4 - A better version of this plugin is on the way, with settings stored in memory and more customization options.
